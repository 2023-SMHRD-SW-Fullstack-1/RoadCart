## 👀 서비스 소개
* 서비스명:  길바구니
* 서비스설명: 지도API를 활용한 여행일정관리 웹서비스
<br>

## 📅 프로젝트 기간
2023.05.12 ~ 2023.06.27 (6주)
<br>

## ⭐ 주요 기능
* SNS회원가입/로그인(카카오, 구글)
* 지도API
* 크롤링
* 드래그앤드롭
* 게시글 작성/수정/삭제
* 댓글 작성/수정/삭제
* 좋아요
